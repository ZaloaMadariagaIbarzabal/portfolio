# Eskuetan — Ropa de segunda mano · Bilbo

Tienda online de ropa de segunda mano con historia, con tienda física en
Bilbo (Botikazar 14). Incluye catálogo con filtros, carrito de la compra,
favoritos, registro de usuario y panel para vender prendas. Interfaz
trilingüe (EU / ES / EN).

## 🌐 Demo en vivo

https://eskuetan2023-zaloamadariaga.netlify.app/

## 🛠️ Tecnologías

HTML5, CSS3, JavaScript (vanilla), internacionalización propia EU/ES/EN.
